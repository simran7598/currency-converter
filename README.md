# 💱 Real-Time Currency Converter

A responsive web application that converts currencies using real-time exchange rate data from an API.

---

## 🚀 Features

- 🌍 Real-time currency conversion
- 🔄 Swap currencies functionality
- 📊 Multi-currency results (USD, EUR, GBP, JPY, etc.)
- 🏳️ Country flags for better UI
- ⏱️ Last updated exchange rate time
- ⚠️ Error handling for API failures
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- REST API (ExchangeRate API)
- JSON
